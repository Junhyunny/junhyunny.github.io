## HTTP 란?

> Wiki<br>
> HTTP(HyperText Transfer Protocol)는 WWW(W3) 상에서 정보를 주고받을 수 있는 프로토콜이다. 주로 HTML 문서를 주고받는 데에 쓰인다.

서버와 HTML(HyperText Markup Language) 문서를 주고 받는데 사용하는 프로토콜입니다. 
브라우저는 HTML 문서를 해석하여 사용자가 볼 수 있는 웹 페이지를 창에 띄어줍니다.
